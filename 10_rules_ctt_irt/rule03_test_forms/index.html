    规则3：可互换的测验形式¶

    规则对比

    CTT IRT
    当测验形式是平行的时，在多种测验中比较得分是最优的 当不同测验的难度水平因人而异时，在多种测验中比较得分是最优的

    1. 核心概念澄清¶

    1.1 什么是测验形式？¶

    测验形式（Test Forms）

    定义：测量同一种能力的不同版本测验

    现实例子：

    • 高考每年有不同试卷，但都测同一学科能力
    • SAT每次考试题目不同，但要保证分数可比
    • 托福机考每人题目不同，但分数要公平
    • 驾照考试有题库，每人抽到不同题目

    核心问题：如何保证不同版本的分数可以比较？

    1.2 什么是平行测验？¶

    平行测验（Parallel Tests）

    直观理解：两个测验"完全等价"，可以互相替代

    严格要求：

    • 测量相同的心理特质
    • 具有相同的统计属性
    • 对每个人都一样可靠

    2. 平行测验的严格理论¶

    2.1 Gulliksen 的三个条件¶

    设两个测验 \(X\) 和 \(Y\) 分别测量同一被试的同一心理属性，其观测分数：

    \[X = T_X + E_X\]
    \[Y = T_Y + E_Y\]

    其中 \(T_X, T_Y\) 为各自测验的真分数，\(E_X, E_Y\) 为测量误差。

    平行测验的三个必要充分条件

    条件1：相等的真分数

    \[T_X = T_Y\]

    即每个被试在两个测验中的真分数完全相同。

    条件2：相等的误差方差

    \[\operatorname{Var}(E_X) = \operatorname{Var}(E_Y)\]

    两个测验的测量误差具有相同的变异程度。

    条件3：独立的误差

    \[\operatorname{Cov}(E_X, E_Y) = 0\]

    两个测验的误差项彼此不相关。

    2.2 推导平行测验的性质¶

    性质1：均值相等

    \[\begin{align} \mu_X &= \mathbb{E}[X] = \mathbb{E}[T_X + E_X] \ &= \mathbb{E}[T_X] + \mathbb{E}[E_X] \ &= \mathbb{E}[T_Y] + 0 \ &= \mathbb{E}[Y] = \mu_Y \end{align}\]

    性质2：方差相等

    \[\begin{align} \sigma_X^2 &= \operatorname{Var}(X) = \operatorname{Var}(T_X + E_X) \ &= \operatorname{Var}(T_X) + \operatorname{Var}(E_X) + 2\operatorname{Cov}(T_X, E_X) \ &= \sigma_T^2 + \sigma_E^2 + 0 \ &= \sigma_Y^2 \end{align}\]

    其中使用了：

    • \(\operatorname{Cov}(T_X, E_X) = 0\)（CTT基本假设）
    • \(T_X = T_Y\)（条件1）
    • \(\operatorname{Var}(E_X) = \operatorname{Var}(E_Y)\)（条件2）

    性质3：相关等于信度

    \[\begin{align} r_{XY} &= \frac{\operatorname{Cov}(X, Y)}{\sigma_X \sigma_Y} \ &= \frac{\operatorname{Cov}(T_X + E_X, T_Y + E_Y)}{\sigma_X^2} \ &= \frac{\operatorname{Cov}(T_X, T_Y) + \operatorname{Cov}(T_X, E_Y) + \operatorname{Cov}(E_X, T_Y) + \operatorname{Cov}(E_X, E_Y)}{\sigma_X^2} \ &= \frac{\sigma_T^2 + 0 + 0 + 0}{\sigma_X^2} \ &= \frac{\sigma_T^2}{\sigma_X^2} = r_{XX} = r_{YY} \end{align}\]

    平行测验的统计特征总结

    \[\boxed{\mu_X = \mu_Y, \quad \sigma_X = \sigma_Y, \quad r_{XY} = r_{XX} = r_{YY}}\]

    含义：

    • 相同的平均难度
    • 相同的分数离散程度
    • 测验间相关等于各自的信度

    2.3 为什么需要这些条件？¶

    相等均值的必要性：

    • 如果测验A平均分80，测验B平均分60
    • 那么同一个人在两个测验上的分数不可比

    相等方差的必要性：

    • 如果测验A分数很分散，测验B分数很集中
    • 那么相同分数差异的含义不同

    相等信度的必要性：

    • 如果测验A很可靠，测验B不可靠
    • 那么分数的可信度不同

    3. 等值问题与方法¶

    3.1 什么是等值？¶

    等值（Equating）

    定义：使不同测验形式的分数具有可比性的统计过程

    基本思想：

    找到一个变换函数 \(f\)，使得：

    \[Y = f(X)\]

    其中 \(X\) 是一个测验的分数，\(Y\) 是另一个测验的"等价"分数

    理想目标：

    如果两个人能力相同，那么经过等值后，他们的分数应该相同

    3.2 线性等值¶

    模型：

    \[Y = aX + b\]

    参数确定：

    通过矩匹配：

    \[a = \frac{\sigma_Y}{\sigma_X}\]
    \[b = \mu_Y - a\mu_X\]

    数值例子：

    • 测验A：\(\mu_A = 70, \sigma_A = 15\)
    • 测验B：\(\mu_B = 80, \sigma_B = 12\)

    计算：

    \[a = \frac{12}{15} = 0.8\]
    \[b = 80 - 0.8 \times 70 = 24\]

    因此：\(Y = 0.8X + 24\)

    如果某人测验A得60分：

    \[Y = 0.8 \times 60 + 24 = 72\]

    3.3 非线性等值¶

    等百分位等值¶

    等百分位等值（Equipercentile Equating）

    原理：相同百分位数的分数被认为等价

    步骤：

    1. 计算两个测验的累积分布函数 \(F_X(x)\) 和 \(F_Y(y)\)
    2. 找到等价分数：\(F_X(x) = F_Y(y)\)
    3. 等值函数：\(y = F_Y^{-1}(F_X(x))\)

    为什么需要非线性等值？¶

    当两个测验的分数分布形状不同时，线性等值不够准确：

    • 测验A：正态分布
    • 测验B：偏态分布

    线性变换无法同时保持所有百分位点的对应关系。

    3.4 传统等值的根本问题¶

    CTT等值的局限性

    边界效应：

    • 地板效应：困难测验中，0分可能对应很大的能力范围
    • 天花板效应：容易测验中，满分可能对应很大的能力范围

    信息丢失：

    • 极端分数区域的区分度很低
    • 等值在这些区域变得不可靠

    样本依赖：

    • 等值参数依赖于特定样本
    • 换个样本，等值关系可能改变

    4. IRT的革命性解决方案¶

    4.1 共同量表的优雅性¶

    IRT的核心优势

    统一的能力量表：

    • 所有题目都在同一个 \(\theta\) 量表上定义难度
    • 所有被试都在同一个 \(\theta\) 量表上估计能力
    • 无需等值转换，直接可比

    含义明确：

    \[P(\text{成功}) = \frac{e^{(\theta - b)}}{1 + e^{(\theta - b)}}\]

    \(\theta - b\) 直接决定成功概率

    4.2 自适应测验算法¶

    基本流程¶

    计算机自适应测验（CAT）

    步骤1：初始化

    设定初始能力估计 \(\hat{\theta}_0 = 0\)

    步骤2：选题

    选择在当前估计处信息最大的题目：

    \[i^* = \arg\max_i I_i(\hat{\theta})\]

    步骤3：施测与记录

    获得反应 \(u_i \in \{0, 1\}\)

    步骤4：更新估计

    使用最大似然估计更新能力

    Newton-Raphson 更新算法¶

    对数似然函数：

    \[\ln L(\theta) = \sum_{i \in \text{已答}} [u_i \ln P_i(\theta) + (1-u_i) \ln(1-P_i(\theta))]\]

    一阶导数（得分函数）：

    \[S(\theta) = \frac{d \ln L}{d\theta} = \sum_{i \in \text{已答}} [u_i - P_i(\theta)]\]

    二阶导数（信息函数的负值）：

    \[\frac{d^2 \ln L}{d\theta^2} = -\sum_{i \in \text{已答}} P_i(\theta)[1-P_i(\theta)] = -I(\theta)\]

    Newton-Raphson 迭代公式

    \[\hat{\theta}_{n+1} = \hat{\theta}_n - \frac{S(\hat{\theta}_n)}{-I(\hat{\theta}_n)} = \hat{\theta}_n + \frac{\sum_i [u_i - P_i(\hat{\theta}_n)]}{\sum_i P_i(\hat{\theta}_n)[1-P_i(\hat{\theta}_n)]}\]

    直观理解：

    • 分子：观察与预期的偏差
    • 分母：当前的信息量
    • 调整幅度 = 偏差/信息

    4.3 自适应的优势分析¶

    信息效率对比¶

    固定测验：

    \[I_{\text{fixed}}(\theta) = \sum_{i=1}^n I_i(\theta)\]

    其中很多 \(I_i(\theta)\) 接近0（题目太难或太易）

    自适应测验：

    \[I_{\text{adaptive}}(\theta) = \sum_{i=1}^n \max_j I_j(\theta)\]

    每个题目都选择当前最优的

    定理：

    \[I_{\text{adaptive}}(\theta) \geq I_{\text{fixed}}(\theta) \quad \forall \theta\]

    当且仅当固定测验恰好都是最优题目时等号成立（几乎不可能）。

    实证效果¶

    从第2章图2.5可以看出：

    • 固定测验（困难）：\(r^2 = 0.8666\)
    • 固定测验（容易）：\(r^2 = 0.8736\)
    • 自适应测验：\(r^2 = 0.9695\)

    关键洞察

    自适应测验不仅节省题目，更重要的是提供了更准确的能力估计！

    5. 实践建议¶

    选择指南

    使用CTT平行测验：

    • 纸笔测验环境
    • 需要预先印制
    • 法规要求"公平性"

    使用IRT自适应测验：

    • 计算机化环境
    • 个性化评估需求
    • 追求测量效率
