# 7. 概念补充13：IRT量表的"温度计"类比

为了更好地理解IRT量表的特性，我们可以用温度计来类比。

## 7.1 温度测量的类比

温度计类比的启示

**摄氏度 vs 华氏度：**

- 同样的温度，不同的数值（0°C = 32°F）
- 转换关系：\(F = 1.8C + 32\)
- 重要关系保持不变（水的沸点仍然比冰点高100度）

**IRT量表的类比：**

- 同样的能力，不同的数值（不同锚定）
- 转换关系：线性变换
- 重要关系保持不变（能力差异的含义）

## 7.2 零点选择的任意性

零点设定的启示

**温度的零点：**

- 摄氏度：以水的冰点为0度
- 华氏度：以某种盐水溶液的冰点为0度
- 开尔文：以绝对零度为0度

**IRT的零点：**

- 题目锚定：以题目平均难度为0
- 人群锚定：以人群平均能力为0
- 其他锚定：任意选择参照点

## 7.3 测量质量的保持

无论选择什么样的零点和单位，测量的本质关系都保持不变：

- 相对位置不变
- 差异的含义不变
- 比较的结果不变
