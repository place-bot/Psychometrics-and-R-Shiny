# 5. 概念补充11：特定客观性原则的深度解析

Rasch模型的一个重要特征是满足"特定客观性"原则，这个概念值得深入理解。

## 5.1 特定客观性的基本概念

特定客观性的定义

**基本思想：** 对象之间的比较必须可以推广超出观察它们的特定条件

**具体含义：** 测量结果不应该依赖于测量的具体条件，包括特定的题目或特定的被试群体

## 5.2 两种不变比较的详细推导

### 5.2.1 人员比较的不变性

**问题：** 两个人的能力比较是否受题目选择影响？

**Rasch模型中的证明：**

对于任意两个人\(p\)和\(q\)，在任意题目\(i\)上：

\(\ln\left(\frac{P_{pi}/(1-P_{pi})}{P_{qi}/(1-P_{qi})}\right) = \ln\left(\frac{P_{pi}}{1-P_{pi}}\right) - \ln\left(\frac{P_{qi}}{1-P_{qi}}\right)\)

\(= (\theta_p - \beta_i) - (\theta_q - \beta_i) = \theta_p - \theta_q\)

这个结果与题目难度\(\beta_i\)无关！

### 5.2.2 题目比较的不变性

**问题：** 两个题目的难度比较是否受被试选择影响？

**Rasch模型中的证明：**

对于任意两个题目\(i\)和\(j\)，在任意被试\(p\)上：

\(\ln\left(\frac{P_{pi}/(1-P_{pi})}{P_{pj}/(1-P_{pj})}\right) = (\theta_p - \beta_i) - (\theta_p - \beta_j) = \beta_j - \beta_i\)

这个结果与被试能力\(\theta_p\)无关！

## 5.3 特定客观性的实际意义

实际应用中的价值

**适应性测验：** 不同被试可以接受不同题目，但能力比较仍然有效

**题目银行：** 可以用不同被试群体来校准题目参数

**纵向研究：** 可以在不同时间点使用不同题目来追踪能力变化
