# 规则5详解：构建有意义的量表得分

规则对比

| CTT | IRT |
| --- | --- |
| 测验得分通过与常模群体的位置比较而获得意义 | 测验得分通过与题目的距离比较而获得意义 |

## 1. 得分意义的本质问题

得分意义（Score Meaning）

**核心问题**：一个测验分数到底代表什么？

**例子**：学生考试得80分

- 这个80分本身没有绝对意义
- 需要参照标准来解释

**两种解释方式**：

1. **相对解释**：80分在班级中排第几名？
2. **绝对解释**：80分说明掌握了哪些知识技能？

## 2. 常模参照解释（CTT方法）

### 2.1 常模的概念

常模（Norm）是特定群体在测验上的表现分布

**常模群体类型**：

- 全国常模：代表全国同龄人水平
- 地区常模：代表某地区同龄人水平
- 学校常模：代表某学校同年级水平
- 专业常模：代表某专业学生水平

### 2.2 常模分数转换

**百分位数**：

\[
P_r = \frac{\text{低于该分数的人数}}{\text{总人数}} \times 100\%
\]

**标准分数（Z-score）**：

\[
z = \frac{X - \mu}{\sigma}
\]

**T分数**：

\[
T = 50 + 10z = 50 + 10 \times \frac{X - \mu}{\sigma}
\]

### 2.3 标准分数的性质

设原始分数 \(X\)，均值 \(\mu_X\)，标准差 \(\sigma_X\)

标准化变换：

\[
Z = \frac{X - \mu_X}{\sigma_X}
\]

验证均值：

\[
E[Z] = E\left[\frac{X - \mu_X}{\sigma_X}\right] = \frac{E[X] - \mu_X}{\sigma_X} = \frac{\mu_X - \mu_X}{\sigma_X} = 0
\]

验证方差：

\[
\text{Var}(Z) = \text{Var}\left(\frac{X - \mu_X}{\sigma_X}\right) = \frac{\text{Var}(X)}{\sigma_X^2} = \frac{\sigma_X^2}{\sigma_X^2} = 1
\]

因此 \(Z \sim N(0,1)\)

### 2.4 FIM量表的常模参照解释

功能独立性测量（FIM）

评估日常生活活动能力，常用于老年人功能评估

**四个被试在80-89岁常模群体中的位置**：

**Anna**：\(z = -1.5\)

- 低于平均水平1.5个标准差
- 百分位数：\(\Phi(-1.5) \approx 0.07 = 7\%\)
- 解释：功能水平明显低于同龄人

**Paul**：\(z = 0\)

- 处于平均水平
- 百分位数：\(\Phi(0) = 0.50 = 50\%\)
- 解释：功能水平与同龄人相当

**Vera**：\(z = +1.0\)

- 高于平均水平1个标准差
- 百分位数：\(\Phi(1.0) \approx 0.84 = 84\%\)
- 解释：功能水平明显高于同龄人

**Mary**：\(z = +1.5\)

- 高于平均水平1.5个标准差
- 百分位数：\(\Phi(1.5) \approx 0.93 = 93\%\)
- 解释：功能水平远高于同龄人

### 2.5 常模参照的局限性

三大问题

1. **缺乏绝对意义**：只知道相对位置，不知道绝对能力
2. **常模依赖**：不同常模群体结果不同
3. **无法指导干预**：不知道具体问题在哪里

## 3. 项目参照解释（IRT方法）

### 3.1 共同量表的概念

IRT的核心思想：人和题目在同一个量表上

**量表特征**：

- 通常范围：\(-3\) 到 \(+3\)
- 数值含义：能力与难度的差距
- 概率解释：基于logistic函数

### 3.2 成功概率计算

核心公式：

\[
P(\text{成功}) = \frac{e^{(\theta - b)}}{1 + e^{(\theta - b)}}
\]

当 \(\theta = b\) 时，\(P = 0.5\)

**概率计算表**：

| \(\theta - b\) | \(e^{(\theta - b)}\) | \(P\) |
| --- | --- | --- |
| -3 | 0.050 | 0.047 |
| -2 | 0.135 | 0.119 |
| -1 | 0.368 | 0.269 |
| 0 | 1.000 | 0.500 |
| +1 | 2.718 | 0.731 |
| +2 | 7.389 | 0.881 |
| +3 | 20.086 | 0.953 |

### 3.3 FIM项目难度层级

项目难度排序（从易到难）：

1. 梳洗：\(b \approx -1.5\)
2. 轮椅移动：\(b \approx -1.0\)
3. 床/椅转移：\(b \approx 0\)
4. 步行：\(b \approx +0.5\)
5. 爬楼梯：\(b \approx +1.5\)

### 3.4 个体能力的项目参照解释

**Anna**（\(\theta = -1.5\)）：

梳洗：

\[
P = \frac{e^{(-1.5-(-1.5))}}{1+e^0} = 0.5
\]

轮椅移动：

\[
P = \frac{e^{(-1.5-(-1.0))}}{1+e^{-0.5}} = \frac{e^{-0.5}}{1+e^{-0.5}} \approx 0.38
\]

床/椅转移：

\[
P = \frac{e^{(-1.5-0)}}{1+e^{-1.5}} = \frac{e^{-1.5}}{1+e^{-1.5}} \approx 0.18
\]

**解释**：Anna有50%把握完成梳洗，但更复杂的活动成功率很低

**Paul**（\(\theta = 0\)）：

梳洗：

\[
P = \frac{e^{(0-(-1.5))}}{1+e^{1.5}} = \frac{e^{1.5}}{1+e^{1.5}} \approx 0.82
\]

轮椅移动：

\[
P = \frac{e^{(0-(-1.0))}}{1+e^{1.0}} = \frac{e^{1.0}}{1+e^{1.0}} \approx 0.73
\]

床/椅转移：

\[
P = \frac{e^{(0-0)}}{1+e^0} = 0.5
\]

**解释**：Paul能很好完成基本活动，床椅转移有50%把握

### 3.5 项目参照的优势

四大优势

1. **绝对意义明确**：知道具体能做什么，成功概率可量化
2. **个性化诊断**：确定能力边界，找出问题所在
3. **指导干预**：制定训练目标，选择合适难度
4. **跨群体比较**：不依赖特定常模，意义保持一致

## 4. 两种方法的兼容性

### 4.1 IRT分数的线性变换

IRT能力分数 \(\theta\) 可以线性变换为标准分数：

\[
T = 50 + 10\theta
\]

对项目难度进行相应变换：

\[
b' = 50 + 10b
\]

保持了项目参照意义，又可进行常模比较。

### 4.2 应用建议

不同场景的选择

- **临床诊断**：优先使用项目参照解释
- **教育选拔**：可结合常模参照解释
- **能力发展**：重点关注项目参照解释
- **政策制定**：需要常模参照作补充
