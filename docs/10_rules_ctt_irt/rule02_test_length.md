# 规则2：测验长度与信度

规则对比

| CTT | IRT |
| --- | --- |
| 较长的测验比短测验更可靠 | 较短的测验可能比长测验更可靠 |

## 1. CTT：Spearman-Brown 公式

### 1.1 问题设定

核心问题

如果把测验长度增加到原来的 \(n\) 倍，新的信度会是多少？

### 1.2 推导过程

**设定变量**：

- 原测验：长度 \(k\)，信度 \(r_{tt}\)
- 新测验：长度 \(nk\)，信度 \(r_{nn}\)（待求）

**步骤1：方差变化分析**

原测验：

\[\sigma_X^2 = \sigma_T^2 + \sigma_E^2\]

当长度增加 \(n\) 倍时：

- 真分数方差：\(\sigma_{T_n}^2 = n^2\sigma_T^2\)（完全相关，线性叠加）
- 误差方差：\(\sigma_{E_n}^2 = n\sigma_E^2\)（独立误差，方差叠加）

**步骤2：新测验方差**

\[\sigma_{X_n}^2 = n^2\sigma_T^2 + n\sigma_E^2 = n(n\sigma_T^2 + \sigma_E^2)\]

**步骤3：新信度计算**

\[r_{nn} = \frac{\sigma_{T_n}^2}{\sigma_{X_n}^2} = \frac{n^2\sigma_T^2}{n(n\sigma_T^2 + \sigma_E^2)} = \frac{n\sigma_T^2}{n\sigma_T^2 + \sigma_E^2}\]

**步骤4：代入原信度**

由 \(r_{tt} = \frac{\sigma_T^2}{\sigma_T^2 + \sigma_E^2}\) 得：

- \(\sigma_T^2 = r_{tt}(\sigma_T^2 + \sigma_E^2)\)
- \(\sigma_E^2 = (1-r_{tt})(\sigma_T^2 + \sigma_E^2)\)

设 \(V = \sigma_T^2 + \sigma_E^2\)，代入：

\[r_{nn} = \frac{n \cdot r_{tt} \cdot V}{n \cdot r_{tt} \cdot V + (1-r_{tt}) \cdot V} = \frac{nr_{tt}}{nr_{tt} + (1-r_{tt})}\]

Spearman-Brown 公式

\[r_{nn} = \frac{nr_{tt}}{1 + (n-1)r_{tt}}\]

**特殊情况**：

- \(n=2\)：\(r_{22} = \frac{2r_{tt}}{1 + r_{tt}}\)
- \(n=0.5\)：\(r_{0.5} = \frac{0.5r_{tt}}{1 - 0.5r_{tt}}\)

### 1.3 单调性证明

\[\frac{dr_{nn}}{dn} = \frac{r_{tt}(1-r_{tt})}{[1 + (n-1)r_{tt}]^2} > 0\]

因为 \(0 < r_{tt} < 1\)，所以导数恒正，**信度随长度单调递增**。

### 1.4 数值示例

原测验信度 \(r_{tt} = 0.6\)：

| \(n\) | 计算过程 | \(r_{nn}\) |
| --- | --- | --- |
| 2 | \(\frac{2 \times 0.6}{1 + 0.6} = \frac{1.2}{1.6}\) | 0.75 |
| 3 | \(\frac{3 \times 0.6}{1 + 2 \times 0.6} = \frac{1.8}{2.2}\) | 0.82 |
| 0.5 | \(\frac{0.5 \times 0.6}{1 - 0.5 \times 0.6} = \frac{0.3}{0.7}\) | 0.43 |

## 2. IRT：质量胜于数量

### 2.1 信息函数视角

核心概念：测验信息函数

\[I(\theta) = \sum_{i=1}^n I_i(\theta)\]

测验信息 = 所有项目信息之和

### 2.2 固定测验 vs 自适应测验

**固定测验的问题**：

对于能力 \(\theta\) 的被试，项目 \(i\) 的信息：

\[I_i(\theta) = P_i(\theta)[1-P_i(\theta)]\]

当 \(P_i(\theta) \approx 1\)（太简单）或 \(P_i(\theta) \approx 0\)（太难）时，\(I_i(\theta) \approx 0\)

**自适应测验的优势**：

选择使 \(P_i(\theta) \approx 0.5\) 的项目，每个项目都提供最大信息 \(I_i(\theta) = 0.25\)

### 2.3 数值对比

**场景**：能力 \(\theta = 1\) 的被试

**测验A（30道固定题）**：

- 10道简单题：\(P \approx 0.9\)，\(I \approx 0.09 \times 10 = 0.9\)
- 10道中等题：\(P \approx 0.7\)，\(I \approx 0.21 \times 10 = 2.1\)
- 10道困难题：\(P \approx 0.3\)，\(I \approx 0.21 \times 10 = 2.1\)
- 总信息：\(I_A = 5.1\)

**测验B（20道自适应题）**：

- 20道最优题：\(P \approx 0.5\)，\(I \approx 0.25 \times 20 = 5.0\)
- 总信息：\(I_B = 5.0\)

关键发现

20道自适应题目的测量精度 ≈ 30道固定题目！

\[SE_A = \frac{1}{\sqrt{5.1}} = 0.443\]

\[SE_B = \frac{1}{\sqrt{5.0}} = 0.447\]

### 2.4 图形分析：标准误的U形曲线

```text
标准误
  ^
  |     固定20题
  |    /
  |   /
  |  /            \     固定30题
  | /              \   /---------
  |/                \ /
  |                  X
  |              自适应20题
  |            ----------------
  |                              自适应30题
  +-------------------------------------> θ
 -3                  0                  3
```

**U形曲线的解释**：

对于固定测验，总信息函数：

\[I(\theta) = \sum_{i=1}^n I_i(\theta)\]

当所有项目难度集中在 \(b = 0\) 附近时：

\[I(\theta) \approx n \cdot \frac{e^{-\theta^2}}{(1 + e^{-\theta})^2}\]

这在 \(\theta = 0\) 处达到峰值，向两端快速下降，导致：

\[SE(\theta) = \frac{1}{\sqrt{I(\theta)}}\]

在两端迅速上升，形成U形。

## 3. 实践意义

选择建议

**使用 CTT + 增加长度**：

- 资源充足
- 固定纸笔测验
- 群体施测

**使用 IRT + 自适应测验**：

- 计算机化测验
- 个体化测评
- 时间有限但精度要求高
