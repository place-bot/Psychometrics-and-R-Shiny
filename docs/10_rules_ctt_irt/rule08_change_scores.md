# 规则8详解：变化得分的意义

规则对比

| CTT | IRT |
| --- | --- |
| 当初始得分水平不同时，变化得分无法进行有意义的比较 | 即使初始得分水平不同，变化得分也可以进行有意义的比较 |

## 1. 变化得分的定义与重要性

变化得分（Change Score）

测量个体在干预前后的能力变化

**基本公式**：

\[X_{j,\text{change}} = X_{j2} - X_{j1}\]

其中：

- \(X_{j1}\)：个体 \(j\) 的前测分数（干预前）
- \(X_{j2}\)：个体 \(j\) 的后测分数（干预后）
- \(X_{j,\text{change}}\)：个体 \(j\) 的变化得分

**典型应用场景**：

- **教育研究**：教学干预前后的学习效果
- **临床心理学**：治疗前后的症状变化
- **康复医学**：训练前后的功能改善
- **组织心理学**：培训前后的绩效提升

**为什么变化得分的测量如此困难？**

根本原因：变化得分涉及两次测量，误差会累积，而且基线水平的差异会影响变化的可解释性。

## 2. CTT中变化得分的三大根本问题

### 2.1 悖论性可靠性

Bereiter（1963）悖论

前后测相关越低，变化得分可靠性越高！

**完整推导**：

设定：

- \(X_1 = T_1 + E_1\)（前测）
- \(X_2 = T_2 + E_2\)（后测）
- \(D = X_2 - X_1\)（变化得分）
- \(T_D = T_2 - T_1\)（真实变化）

变化得分的可靠性：

\[
r_{DD} = \frac{\sigma_{T_D}^2}{\sigma_D^2}
\]

**步骤1：计算变化得分方差**

\[
\sigma_D^2 = \text{Var}(X_2 - X_1) = \sigma_1^2 + \sigma_2^2 - 2\sigma_{12}
\]

**步骤2：计算真实变化方差**

\[
\sigma_{T_D}^2 = \text{Var}(T_2 - T_1) = \sigma_{T_1}^2 + \sigma_{T_2}^2 - 2\sigma_{T_1 T_2}
\]

**步骤3：表达观察分数的协方差**

\[
\sigma_{12} = \text{Cov}(T_1 + E_1, T_2 + E_2) = \text{Cov}(T_1, T_2) = \sigma_{T_1 T_2}
\]

（假设误差与真分数不相关，且不同时点的误差不相关）

**步骤4：最终可靠性公式**

\[
r_{DD} = \frac{\sigma_{T_1}^2 + \sigma_{T_2}^2 - 2\sigma_{T_1 T_2}}{\sigma_1^2 + \sigma_2^2 - 2\sigma_{12}}
\]

**关键洞察**：当 \(\sigma_{T_1 T_2}\) 接近 \(\sigma_{T_1}\sigma_{T_2}\)（即真分数高度相关）时：

- 分子 \(\sigma_{T_1}^2 + \sigma_{T_2}^2 - 2\sigma_{T_1 T_2}\) 趋向于0
- 分母 \(\sigma_1^2 + \sigma_2^2 - 2\sigma_{12}\) 减小但不趋向于0
- 整个比值 \(r_{DD}\) 趋向于0

### 2.2 虚假负相关

**相关系数计算**：

\[
r(D, X_1) = r(X_2 - X_1, X_1) = \frac{\text{Cov}(X_2 - X_1, X_1)}{\sigma_D \sigma_1}
\]

**展开协方差**：

\[
\text{Cov}(X_2 - X_1, X_1) = \text{Cov}(X_2, X_1) - \text{Cov}(X_1, X_1) = \sigma_{12} - \sigma_1^2
\]

**最终公式**：

\[
r(D, X_1) = \frac{\sigma_{12} - \sigma_1^2}{\sigma_D \sigma_1}
\]

**负相关的条件**：

当 \(\sigma_{12} < \sigma_1^2\) 时，即 \(r_{12} < \frac{\sigma_1}{\sigma_2}\) 时，相关为负。

特殊情况：如果 \(\sigma_1 = \sigma_2\)，则当 \(r_{12} < 1\) 时，相关必然为负！

**回归效应的证明**：

设 \(X_2 = \alpha + \beta X_1 + \epsilon\)，其中 \(\epsilon\) 是随机误差。

则：

\[
D = X_2 - X_1 = \alpha + (\beta - 1)X_1 + \epsilon
\]

当 \(\beta < 1\) 时（回归效应），\(D\) 与 \(X_1\) 必然负相关。

### 2.3 不同初始水平的不同含义

考虑一个100分制的认知能力测验：

- **学生A**：从30分提高到35分（+5分）
- **学生B**：从80分提高到85分（+5分）

**分析1：相对改善幅度**

- 学生A：\((35-30)/30 = 16.7\%\) 的相对改善
- 学生B：\((85-80)/80 = 6.25\%\) 的相对改善

**分析2：距离最大值的改善**

- 学生A：在剩余70分中改善了5分，改善率 \(5/70 = 7.1\%\)
- 学生B：在剩余20分中改善了5分，改善率 \(5/20 = 25\%\)

**分析3：能力分布位置**

如果分数分布呈正态分布，\(\mu = 50, \sigma = 15\)：

- 学生A：从第 \((30-50)/15 = -1.33\) 个标准差改善到 \(-1.0\) 个标准差
- 学生B：从第 \((80-50)/15 = +2.0\) 个标准差改善到 \(+2.33\) 个标准差

在正态分布的尾部，相同的分数差异代表更大的能力差异。

## 3. 为什么变化得分需要区间量表？

### 3.1 区间量表的严格定义

区间量表（Interval Scale）

**定义**：存在函数 \(f: \Theta \to \mathbb{R}\)，使得对于任意 \(\theta_1, \theta_2, \theta_3, \theta_4 \in \Theta\)：

\[
\theta_1 - \theta_2 = \theta_3 - \theta_4 \Leftrightarrow f(\theta_1) - f(\theta_2) = f(\theta_3) - f(\theta_4)
\]

**直观含义**：相等的特质差异对应相等的量表差异，无论在量表的哪个位置

**允许的变换**：\(Y = aX + b \quad (a > 0)\)

**变化得分的要求**：只有在区间量表上，我们才能说：

\[\Delta_1 = \Delta_2 \Leftrightarrow \text{特质变化量相等}\]

### 3.2 CTT难以满足区间量表要求的原因

**原因1：总分的非线性本质**

CTT总分是项目分数的简单相加：

\[
X = \sum_{i=1}^n X_i
\]

但特质水平与项目反应的关系通常是非线性的（如S形曲线）。

**原因2：分数压缩效应**

假设特质 \(\theta\) 与观察分数 \(X\) 的关系为：

\[
X = g(\theta)
\]

其中 \(g\) 是非线性函数。

变化得分为：

\[
\Delta X = g(\theta_2) - g(\theta_1)
\]

只有当 \(g\) 是线性函数时，\(\Delta X\) 才与 \(\Delta \theta\) 成正比。

**例子：逻辑斯蒂函数**

\[
X = \frac{100}{1 + e^{-(\theta - \mu)}}
\]

导数分析：

\[
\frac{dX}{d\theta} = \frac{100e^{-(\theta - \mu)}}{(1 + e^{-(\theta - \mu)})^2}
\]

在 \(\theta = \mu\) 处导数最大，意味着相同的 \(\Delta \theta\) 在中间部分产生最大的 \(\Delta X\)。

## 4. IRT中变化得分的革命性改进

### 4.1 IRT的根本优势

**优势1：真正的区间量表**

在Rasch模型中，对数几率是 \(\theta\) 的线性函数：

\[
\ln\left(\frac{P(\theta)}{1-P(\theta)}\right) = \theta - b
\]

这保证了：

\[
\Delta \theta_1 = \Delta \theta_2 \Leftrightarrow \text{对所有项目的对数几率变化相等}
\]

**优势2：个体化的测量精度**

IRT为每个能力水平提供不同的标准误：

\[
\text{SE}(\hat{\theta}) = \frac{1}{\sqrt{I(\theta)}}
\]

其中 \(I(\theta)\) 是该能力水平的信息函数。

**优势3：样本无关的参数**

IRT参数不依赖于特定样本，变化得分的含义在不同群体中保持一致。

### 4.2 学习与变化的多维Rasch模型（MRMLC）

MRMLC模型

**核心思想**：将个体的能力分解为两个独立成分

**稳定能力**：不受短期干预影响的基础能力

**可修改性**：通过学习或训练可以改变的能力成分

**基本模型**：

\[\theta_{\text{post}} = \theta_{\text{pre}} + \alpha \cdot \text{Modifiability}\]

其中 \(\alpha\) 是干预强度参数

**前测模型**：

\[
P_{i,\text{pre}}(\theta_{\text{pre}}) = \frac{\exp(\theta_{\text{pre}} - b_i)}{1 + \exp(\theta_{\text{pre}} - b_i)}
\]

**后测模型**：

\[
P_{i,\text{post}}(\theta_{\text{post}}) = \frac{\exp(\theta_{\text{post}} - b_i)}{1 + \exp(\theta_{\text{post}} - b_i)}
\]

**能力关系**：

\[
\theta_{\text{post}} = \theta_{\text{pre}} + \Delta_{\text{learned}}
\]

**可修改性建模**：

\[
\Delta_{\text{learned}} = \lambda \cdot M
\]

其中 \(M\) 是个体的可修改性参数，\(\lambda\) 是干预效果参数。

### 4.3 认知老化研究的实例分析

Embretson (1998b) 研究了年轻人和老年人在空间能力训练中的表现差异。

**传统分析**：

\[
\text{原始增益} = X_{\text{post}} - X_{\text{pre}}
\]

**MRMLC分析**：

同时估计稳定能力和可修改性，得到纯粹的学习能力指标。

**为什么关系是非线性的？**

**原因1：基线差异调整**

MRMLC调整了初始能力水平的影响：

\[
\text{原始增益} = f(\text{可修改性}, \theta_{\text{pre}})
\]

**原因2：学习曲线的非线性**

实际的学习过程往往遵循对数或幂函数规律：

\[
\Delta_{\text{learned}} = \lambda \cdot M^{\gamma}
\]

其中 \(\gamma < 1\) 反映了边际递减效应。

## 5. 变化得分分析的实践指南

基于IRT的变化得分分析流程

**前期设计**

- 确保前后测使用相同或等值的测验
- 设计足够的题目数量以获得稳定估计
- 考虑练习效应和记忆效应

**模型选择**

- 简单情况：使用基本IRT模型估计 \(\theta\) 值
- 复杂情况：考虑MRMLC或其他变化模型
- 验证模型假设和拟合度

**参数估计**

- 联合估计前后测参数（确保参数等值）
- 检查参数估计的稳定性
- 计算变化得分的标准误

**结果解释**

- 报告能力量表上的变化（而非原始分数）
- 提供变化的置信区间
- 解释变化的实际意义

**常见陷阱与避免方法**：

**陷阱1：忽略测量误差**

- 问题：将变化得分当作没有误差的真值
- 解决：报告变化的标准误和置信区间

**陷阱2：回归效应**

- 问题：错误归因回归效应为真实变化
- 解决：使用控制组或基线校正方法

**陷阱3：量表依赖**

- 问题：在不同量表上比较变化得分
- 解决：确保使用具有区间量表性质的IRT量表
