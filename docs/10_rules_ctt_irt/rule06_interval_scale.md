# 规则6详解：建立量表属性

规则对比

| CTT | IRT |
| --- | --- |
| 区间量表属性通过获得正态分数分布而实现 | 区间量表属性通过应用正当化的测量模型而实现 |

## 1. 测量量表的四个层次

Stevens（1946）测量量表分类

**名义量表（Nominal Scale）**

- 只能分类，无序关系
- 例：性别、民族、血型
- 允许运算：\(=\), \(\neq\)

**顺序量表（Ordinal Scale）**

- 可以排序，但间距不等
- 例：满意度、疼痛等级
- 允许运算：\(=\), \(\neq\), \(>\), \(<\)

**区间量表（Interval Scale）**

- 间距相等，但无绝对零点
- 例：温度（摄氏度）、标准化测验分数
- 允许运算：\(+\), \(-\)（差值有意义）

**比率量表（Ratio Scale）**

- 有绝对零点，可以比较倍数
- 例：身高、体重、反应时间
- 允许运算：\(\times\), \(\div\)（比值有意义）

## 2. 为什么区间量表重要？

### 2.1 区间量表的核心性质

等距性质

对于区间量表，必须满足：

\[
d(a,b) = d(c,d) \Leftrightarrow |a-b| = |c-d|
\]

其中 \(d(\cdot,\cdot)\) 表示距离函数

**线性变换不变性**：

如果 \(X\) 是区间量表，则 \(Y = aX + b\) （\(a > 0\)）也是区间量表

证明：

\[
|Y_1 - Y_2| = |(aX_1 + b) - (aX_2 + b)| = a|X_1 - X_2|
\]

相对距离保持不变：

\[
\frac{|Y_1 - Y_2|}{|Y_3 - Y_4|} = \frac{a|X_1 - X_2|}{a|X_3 - X_4|} = \frac{|X_1 - X_2|}{|X_3 - X_4|}
\]

### 2.2 支持参数统计的必要性

只有区间量表（及以上）才能合理使用：

- 均值：\(\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i\)
- 方差：\(\sigma^2 = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^2\)
- 相关系数：\(r = \frac{\sum(X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum(X_i - \bar{X})^2\sum(Y_i - \bar{Y})^2}}\)

## 3. CTT实现区间量表的尝试

### 3.1 方法1：项目选择策略

Gulliksen（1950）的策略

选择通过率 \(p \approx 0.5\) 的项目

**理论推导**：

设真分数 \(T \sim N(\mu_T, \sigma_T^2)\)

对于二值计分项目 \(i\)，定义项目特征函数：

\[
u_i = \begin{cases}
1 & \text{if } T > \tau_i \
0 & \text{if } T \leq \tau_i
\end{cases}
\]

其中 \(\tau_i\) 是项目 \(i\) 的阈值。

通过率：

\[
p_i = P(u_i = 1) = P(T > \tau_i) = 1 - \Phi\left(\frac{\tau_i - \mu_T}{\sigma_T}\right)
\]

当 \(p_i = 0.5\) 时：

\[
\Phi\left(\frac{\tau_i - \mu_T}{\sigma_T}\right) = 0.5
\]

因此 \(\tau_i = \mu_T\)

**中心极限定理的应用**：

总分 \(S = \sum_{i=1}^n u_i\)

当所有项目的 \(p_i \approx 0.5\) 时，根据Lindeberg-Feller中心极限定理：

\[
\frac{S - n \cdot 0.5}{\sqrt{n \cdot 0.5 \cdot 0.5}} \xrightarrow{d} N(0,1)
\]

### 3.2 方法2：百分位匹配标准化

非线性变换

**百分位匹配过程**：

1. 计算累积分布函数：\(F_X(x) = P(X \leq x)\)
2. 标准化变换：\(Z = \Phi^{-1}(F_X(X))\)

**非线性变换的证明**：

设变换函数 \(g(x) = \Phi^{-1}(F_X(x))\)

计算导数：

\[
g'(x) = \frac{d}{dx}\Phi^{-1}(F_X(x)) = \frac{f_X(x)}{\phi(\Phi^{-1}(F_X(x)))}
\]

其中 \(f_X(x)\) 是 \(X\) 的密度函数，\(\phi\) 是标准正态密度。

只有当 \(\frac{f_X(x)}{\phi(\Phi^{-1}(F_X(x)))}\) 是常数时，\(g\) 才是线性的。

这要求 \(X\) 本身就是正态分布！

### 3.3 分数压缩的定量分析

分数压缩（Score Compression）

测验分数在某个区间内的分辨率降低

**信息函数视角**：

在CTT中，测验信息函数近似为：

\[
I(T) \approx \sum_{i=1}^n p_i(T)(1-p_i(T))
\]

对于容易测验（\(p_i(T)\) 普遍较高）：

- 当 \(T\) 很高时，\(p_i(T) \approx 1\)，因此 \(I(T) \approx 0\)
- 高分段信息量极低，导致分数压缩

对于困难测验（\(p_i(T)\) 普遍较低）：

- 当 \(T\) 很低时，\(p_i(T) \approx 0\)，因此 \(I(T) \approx 0\)
- 低分段信息量极低，导致分数压缩

## 4. Jones（1971）的区间量表理论

### 4.1 理论框架

Jones的两个假设

1. 真分数 \(T\) 具有区间量表属性
2. 真分数在目标人群中呈正态分布：\(T \sim N(\mu_T, \sigma_T^2)\)

**定理**：在上述假设下，当且仅当观察分数呈正态分布时，观察分数具有区间量表属性。

**证明**：

设观察分数 \(X = T + E\)，其中 \(E\) 是误差。

如果 \(X\) 具有区间量表属性，则存在变换 \(h\) 使得：

\[
h(X) = aT + b
\]

由于 \(T \sim N(\mu_T, \sigma_T^2)\)，所以 \(h(X) \sim N(a\mu_T + b, a^2\sigma_T^2)\)

要使 \(h(X)\) 呈正态分布，\(h\) 必须是线性变换。

因此：\(X = \frac{1}{a}h(X) - \frac{b}{a} = T\)（忽略误差）

这要求 \(X\) 本身就呈正态分布。\(\square\)

### 4.2 理论的局限性

三个主要问题

1. **人群依赖性**：量表属性与特定人群绑定
2. **不可验证性**：真分数不可观察
3. **循环论证**：用正态分布证明区间性质，又用区间性质支持正态化

## 5. IRT的解决方案

### 5.1 Rasch模型的测量理论基础

基本测量原则

**不变比较原理（Invariant Comparison）**：

- 个体比较不依赖于项目
- 项目比较不依赖于个体

**表达式**：

对于两个个体 \(\theta_1\) 和 \(\theta_2\)，在任意项目 \(i\) 上：

\[
\frac{P_i(\theta_1)/[1-P_i(\theta_1)]}{P_i(\theta_2)/[1-P_i(\theta_2)]} = \frac{e^{\theta_1 - b_i}}{e^{\theta_2 - b_i}} = e^{\theta_1 - \theta_2}
\]

这个比值只依赖于 \(\theta_1 - \theta_2\)，与项目参数 \(b_i\) 无关。

对于两个项目 \(i\) 和 \(j\)，在任意个体 \(\theta\) 上：

\[
\frac{P_i(\theta)/[1-P_i(\theta)]}{P_j(\theta)/[1-P_j(\theta)]} = \frac{e^{\theta - b_i}}{e^{\theta - b_j}} = e^{b_j - b_i}
\]

这个比值只依赖于 \(b_j - b_i\)，与个体参数 \(\theta\) 无关。

### 5.2 对数几率的线性结构

Logit变换

\[
\text{logit}(P) = \ln\left(\frac{P}{1-P}\right)
\]

在Rasch模型中：

\[
\text{logit}(P_i(\theta)) = \theta - b_i
\]

**线性性质的证明**：

设两个能力水平 \(\theta_1\) 和 \(\theta_2\)，两个项目 \(i\) 和 \(j\)

差值：

\[
\text{logit}(P_i(\theta_1)) - \text{logit}(P_j(\theta_2)) = (\theta_1 - b_i) - (\theta_2 - b_j) = (\theta_1 - \theta_2) - (b_i - b_j)
\]

这是 \(\theta\) 差值和 \(b\) 差值的线性组合，满足区间量表要求。

### 5.3 充分统计量与区间性质

充分统计量的作用

在Rasch模型中，原始总分 \(r = \sum u_i\) 是能力 \(\theta\) 的充分统计量

**单调变换**：

\[
\hat{\theta} = \ln\left(\frac{r}{n-r}\right) + C
\]

其中 \(C\) 是标准化常数。

这个变换保持了顺序，并赋予了区间性质。

## 6. 实践意义

应用建议

**CTT情境**：

- 小样本研究
- 快速开发的测验
- 已有正态常模的测验

**IRT情境**：

- 大规模标准化测验
- 需要跨群体比较
- 计算机自适应测验

**验证方法**：

- 检查等值函数的线性性
- 验证不同群体的测量不变性
- 分析项目-个体交互作用
