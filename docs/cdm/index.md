# 认知诊断模型（CDM）

认知诊断模型（Cognitive Diagnosis Models, CDM）用于从测验反应中推断被试对多个细粒度属性或技能的掌握状态。与常见 IRT 模型主要估计连续潜在能力不同，CDM 更关注“掌握了哪些属性”和“哪些属性还需要补救”。

## 核心问题

CDM 分区后续适合围绕以下问题组织：

- 属性定义：测验希望诊断哪些知识点、技能或认知过程。
- Q 矩阵：每道题需要哪些属性才能正确作答。
- 掌握模式：被试在多个属性上的掌握与未掌握组合。
- 反应模型：属性掌握状态如何映射到答对概率。
- 模型评价：Q 矩阵、模型拟合、分类准确性和诊断报告是否可信。

## 与 IRT 的关系

| 关注点 | IRT | CDM |
| --- | --- | --- |
| 主要目标 | 估计潜在能力水平 | 诊断属性掌握状态 |
| 潜变量形式 | 通常为连续能力 | 通常为离散属性向量 |
| 结果解释 | 能力高低、项目参数、测验信息 | 掌握/未掌握模式、补救建议 |
| 典型应用 | 测验等值、评分、题库建设 | 诊断测评、学习反馈、精准补救 |
