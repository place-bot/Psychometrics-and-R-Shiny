# 经典测量理论（CTT）与项目反应理论（IRT）

## 0. 前言：历史背景

### 0.1 经典测量理论（CTT）的发展

历史脉络

- **1907-1913年**：Spearman 奠定了经典测量理论的基础
- **1950年**：Gulliksen 出版经典教材，CTT 体系成熟
- **1968年**：Lord 和 Novick 提出项目反应理论（IRT）
- **现在**：IRT 逐渐成为主流，但 CTT 仍广泛使用

### 0.2 核心术语详解

#### 0.2.1 什么是"等值"（Equating）？

等值的定义与应用

**等值**：使不同版本的测验具有可比性的过程

**实际例子**：

- 高考有多套试卷，但最终分数要能相互比较
- SAT 每年都有新题目，但分数标准要保持一致
- 托福机考每个人题目不同，但分数要公平

**传统等值方法**：

1. **线性等值**：\(Y = aX + b\)（简单的线性变换）
2. **非线性等值**：使用更复杂的曲线关系

**为什么需要非线性等值？**

因为不同测验的分数分布可能不同：

- 测验 A：大部分人得中等分数，呈正态分布
- 测验 B：很多人得高分或低分，分布偏斜
- 简单的线性变换无法处理这种差异

#### 0.2.2 什么是"Delta 方法"？

Delta 方法详解

**Delta 方法**：Gulliksen（1950）提出的题目难度指标

**核心公式**：

\[\Delta = 13 + 4\Phi^{-1}(p)\]

其中：

- \(p\) = 题目的通过率（答对的人数比例）
- \(\Phi^{-1}\) = 标准正态分布的逆函数
- 13 和 4 是为了避免负数和小数

**直观理解**：

- 如果 50% 的人答对，\(p=0.5\)，\(\Delta = 13\)（中等难度）
- 如果 90% 的人答对，\(p=0.9\)，\(\Delta\) 较小（简单题目）
- 如果 10% 的人答对，\(p=0.1\)，\(\Delta\) 较大（困难题目）

#### 0.2.3 什么是"区间量表"？

**测量量表的四个层次**：

1. **名义量表**：只能分类（如：男/女）
2. **顺序量表**：能排序但间距不等（如：满意/一般/不满意）
3. **区间量表**：间距相等但没有绝对零点（如：温度）
4. **比率量表**：有绝对零点（如：身高、体重）

区间量表的特征

- 数值间的差距有意义
- 60 分和 70 分的差距 = 70 分和 80 分的差距
- 但不能说 80 分是 40 分的"两倍好"

**心理测量中的问题**：
原始分数通常不满足区间量表的要求，需要通过统计变换来实现。

#### 0.2.4 什么是"常模"？

常模（Norm）

**定义**：参照群体的表现标准

**例子**：

- 某学生 IQ 测试得 100 分
- 这个分数本身没意义
- 但如果常模显示：平均分 = 100，标准差 = 15
- 那么这个学生就是"平均水平"

**常见的常模分数**：

- **百分位数**：超过多少百分比的人
- **标准分数**：\((X-\mu)/\sigma\)
- **T 分数**：\(50 + 10 \times \text{标准分数}\)

#### 0.2.5 什么是"难度平行"？

难度平行的概念与问题

**难度平行**：不同测验版本的整体难度相同

**传统做法**：

- 每个版本都包含同样比例的简单、中等、困难题目
- 平均难度相同
- 分数分布相似

**问题**：
这种做法假设所有人的能力分布相同，但实际情况往往不是这样。

#### 0.2.6 什么是"题干刺激"？

题干刺激的影响

**题干刺激**：题目中影响答题的非核心因素

**例子**：
数学题："小明买了 3 个苹果..."

- 核心：测试计算能力
- 刺激：人名（小明）、物品（苹果）、情境（买东西）

**观点对比**：

- **传统观点**：这些刺激不影响测量
- **现代观点**：刺激可能影响不同群体的表现
  - 城市孩子熟悉"买苹果"的情境
  - 农村孩子可能更熟悉"种苹果"的情境

#### 0.2.7 什么是"得分解释"？

**得分解释**：如何理解和使用测验分数

1. **绝对解释**：分数本身有固定含义
2. 例：考试及格线 60 分
3. **相对解释**：分数相对于他人的表现
4. 例：排名前 10%
5. **个体内解释**：同一个人不同时间的比较
6. 例：这次比上次进步了 5 分

## 1. CTT 基础理论与公式推导

### 1.1 CTT 的基本假设和推导

#### 1.1.1 核心公式的由来

CTT 基本假设

**基本公式**：

\[X = T + E\]

其中：

- \(X\) = 观察分数（实际测到的分数）
- \(T\) = 真分数（理论上的"真实"分数）
- \(E\) = 测量误差（随机波动）

**进一步假设**：

1. \(E(E) = 0\)（误差的期望值为 0）
2. \(\rho(T,E) = 0\)（真分数与误差不相关）
3. \(\rho(E_1,E_2) = 0\)（不同测量的误差不相关）

#### 1.1.2 信度公式的推导

**第一步：分解观察分数的方差**

\[\text{Var}(X) = \text{Var}(T + E) = \text{Var}(T) + \text{Var}(E) + 2\text{Cov}(T,E)\]

由于 \(\text{Cov}(T,E) = 0\)：

\[\sigma_X^2 = \sigma_T^2 + \sigma_E^2\]

**第二步：定义信度**

信度是真分数方差在总方差中的比例：

\[r_{tt} = \frac{\sigma_T^2}{\sigma_X^2} = \frac{\sigma_T^2}{\sigma_T^2 + \sigma_E^2}\]

**第三步：推导标准误公式**

从 \(\sigma_X^2 = \sigma_T^2 + \sigma_E^2\) 可得：

\[\sigma_E^2 = \sigma_X^2 - \sigma_T^2 = \sigma_X^2(1 - \frac{\sigma_T^2}{\sigma_X^2}) = \sigma_X^2(1 - r_{tt})\]

因此：

\[SE_{\text{msmt}} = \sigma_E = \sigma_X\sqrt{1 - r_{tt}}\]

CTT 标准误公式总结

**标准误公式**：

\[SE_{\text{msmt}} = \sigma \sqrt{1 - r_{tt}}\]

**推导逻辑**：

1. 观察分数 = 真分数 + 误差
2. 总方差 = 真分数方差 + 误差方差
3. 信度 = 真分数方差 / 总方差
4. 标准误 = 误差的标准差

### 1.2 Spearman-Brown 公式推导

#### 1.2.1 问题设定

假设我们有一个测验，信度为 \(r_{tt}\)，现在想知道：如果把测验长度增加到原来的 \(n\) 倍，新的信度会是多少？

#### 1.2.2 详细推导过程

**步骤 1：设定变量**

- 原测验：长度为 \(k\)，信度为 \(r_{tt}\)
- 新测验：长度为 \(nk\)，信度为 \(r_{nn}\)（待求）

**步骤 2：分析方差变化**

原测验的方差分解：

\[\sigma_X^2 = \sigma_T^2 + \sigma_E^2\]

当测验长度增加 \(n\) 倍时：

- 真分数方差增加 \(n^2\) 倍：\(\sigma_{T_n}^2 = n^2\sigma_T^2\)
- 误差方差增加 \(n\) 倍：\(\sigma_{E_n}^2 = n\sigma_E^2\)（假设误差独立）

**步骤 3：计算新的总方差**

\[\sigma_{X_n}^2 = \sigma_{T_n}^2 + \sigma_{E_n}^2 = n^2\sigma_T^2 + n\sigma_E^2\]

**步骤 4：计算新的信度**

\[r_{nn} = \frac{\sigma_{T_n}^2}{\sigma_{X_n}^2} = \frac{n^2\sigma_T^2}{n^2\sigma_T^2 + n\sigma_E^2} = \frac{n^2\sigma_T^2}{n(n\sigma_T^2 + \sigma_E^2)}\]

化简：

\[r_{nn} = \frac{n\sigma_T^2}{n\sigma_T^2 + \sigma_E^2}\]

**步骤 5：用原信度表示**

从 \(r_{tt} = \frac{\sigma_T^2}{\sigma_T^2 + \sigma_E^2}\) 可得：

- \(\sigma_T^2 = r_{tt}(\sigma_T^2 + \sigma_E^2)\)
- \(\sigma_E^2 = (1-r_{tt})(\sigma_T^2 + \sigma_E^2)\)

设 \(\sigma_T^2 + \sigma_E^2 = V\)，则：

\[r_{nn} = \frac{n \cdot r_{tt} \cdot V}{n \cdot r_{tt} \cdot V + (1-r_{tt}) \cdot V} = \frac{nr_{tt}}{nr_{tt} + (1-r_{tt})}\]

Spearman-Brown 公式

**最终公式**：

\[r_{nn} = \frac{nr_{tt}}{1 + (n-1)r_{tt}}\]

**特殊情况**：

- \(n=2\)（长度翻倍）：\(r_{22} = \frac{2r_{tt}}{1 + r_{tt}}\)
- \(n=0.5\)（长度减半）：\(r_{0.5} = \frac{0.5r_{tt}}{1 - 0.5r_{tt}}\)

#### 1.2.3 数值例子

假设原测验信度 \(r_{tt} = 0.6\)：

**长度翻倍**（\(n=2\)）：

\[r_{22} = \frac{2 \times 0.6}{1 + (2-1) \times 0.6} = \frac{1.2}{1.6} = 0.75\]

**长度增加 3 倍**（\(n=3\)）：

\[r_{33} = \frac{3 \times 0.6}{1 + (3-1) \times 0.6} = \frac{1.8}{2.2} = 0.82\]

**长度减半**（\(n=0.5\)）：

\[r_{0.5} = \frac{0.5 \times 0.6}{1 + (0.5-1) \times 0.6} = \frac{0.3}{0.7} = 0.43\]

## 2. IRT 基础理论

### 2.1 IRT 的核心思想

IRT 的革命性观点

1. **个体差异**：不同能力水平的人，测量精度不同
2. **项目特征**：每道题都有自己的难度和区分度
3. **模型驱动**：基于概率模型，而非简单统计
4. **不变性**：题目参数不依赖于特定样本

### 2.2 最简单的 IRT 模型：Rasch 模型

#### 2.2.1 模型公式

\[P_i(\theta) = \frac{e^{(\theta - b_i)}}{1 + e^{(\theta - b_i)}}\]

其中：

- \(P_i(\theta)\) = 能力为 \(\theta\) 的人答对第 \(i\) 题的概率
- \(\theta\) = 人的能力参数（特质水平）
- \(b_i\) = 第 \(i\) 题的难度参数
- \(e\) = 自然常数

#### 2.2.2 模型特征

**当 \(\theta = b_i\) 时**：

\[P_i(\theta) = \frac{e^0}{1 + e^0} = \frac{1}{2} = 0.5\]

这意味着：当人的能力等于题目难度时，答对概率是 50%。

**当 \(\theta >> b_i\) 时**：

\[P_i(\theta) \approx 1\]

（能力远高于难度，几乎肯定答对）

**当 \(\theta << b_i\) 时**：

\[P_i(\theta) \approx 0\]

（能力远低于难度，几乎肯定答错）
