# IRT 估计与评估

这一部分放 IRT 模型拟合之后最关键的工作：参数估计（parameter estimation）、特质量表（trait scale）、考生评分（scoring）和模型拟合评估（model fit assessment）。

## 学习顺序

1. [IRT 参数估计](../estimate.md)：理解项目参数如何从反应数据中估计出来。
2. [IRT 特质量表](../scale/index.md)：理解 \(\theta\) 量表的单位、解释和特定客观性（specific objectivity）。
3. [考生评分](../score/index.md)：比较 MLE、MAP、EAP 和分数变换。
4. [模型拟合评估](../assess.md)：检查模型假设是否足够支持解释。

## 这一部分的核心问题

- 项目参数和能力参数是否能稳定估计。
- \(\theta\) 的数值单位到底表示什么。
- 为什么同一个原始分需要通过模型转换才能成为可比较的量表分。
- 项目信息（item information）和测验信息（test information）如何决定不同能力水平上的测量精度。
- 模型拟合不好时，分数解释会在哪些地方失效。

## 输出物

| 输出 | 用途 |
| --- | --- |
| 项目参数 | 描述项目难度、区分度和其他模型参数 |
| 能力估计 | 给被试定位在潜在特质量表上 |
| 标准误 | 表示个体分数的不确定性 |
| 信息函数 | 判断测验在哪些能力区间最精确 |
| 拟合指标 | 判断模型假设和数据是否相容 |
