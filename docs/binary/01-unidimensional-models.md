# 1. 用于二元数据的单维IRT模型

## 1.1 什么时候使用单维IRT模型？

在单维IRT模型中，单一潜在特质被认为足以表征个体差异。

**通俗理解：** 就像用一把尺子测量身高，我们假设一个数值就能代表某种能力。

单维IRT模型适用于以下情况：

1. 当单一共同因素支配项目反应时
2. 当所有项目涉及每个因素的相同组合时（即使有多个潜在因素）

**实例说明：**

- 适用：纯计算能力测验（所有题目都只测计算）
- 适用：即使测验包含计算和推理，但每道题都是70%计算+30%推理的固定组合

单维IRT模型不适用的情况

当以下情况出现时，单维IRT模型不适用：

1. 两个或更多潜在特质对项目有不同影响
2. 个体在应用于项目的策略、知识结构或解释方面存在系统性差异

在这些情况下，应使用多维IRT模型。

**举例：**

- 语文测验中，有些题目主要考阅读理解，有些主要考写作能力
- 学生解题时，有人用代数方法，有人用几何方法

## 1.2 符号系统

在本章介绍的单维模型中，将使用以下符号：

- \(X_{is}\)：个体 \(s\) 对项目 \(i\) 的反应（0 或 1）
- \(\theta_s\)：个体 \(s\) 的特质水平（能力值）
- \(\beta_i\)：项目 \(i\) 的难度
- \(\alpha_i\)：项目 \(i\) 的区分度
- \(\gamma_i\)：项目 \(i\) 的下渐近线（猜测参数）

**直观理解这些参数：**

- \(\theta_s\)：就像学生的"能力分数"，通常在-3到+3之间
- \(\beta_i\)：题目的"难度分数"，与能力使用相同尺度
- \(\alpha_i\)：题目"敏感度"，能否区分不同能力的学生
- \(\gamma_i\)：纯猜对的概率下限

## 1.3 示例数据集：抽象推理测验（ART）

为了说明三个逻辑单维IRT模型，我们使用一个大型数据集。数据来自抽象推理测验（ART；Embretson，1998），包含：

- 30个项目
- 787名年轻成人样本
- 使用BILOG程序通过边际最大似然法估计IRT参数

观察数据特点：

- 项目难度差异很大：正确百分比从20.8%到94.8%
- 双列相关全部为正值，只有少数低于.30

**数据解读：**

- 项目1几乎所有人都答对（94.8%），说明很简单

- 双列相关高的项目（如项目1的.582）能很好地区分高低能力者

## 1.4 双列相关（Point-Biserial Correlation）

**定义**：

双列相关是一种特殊的皮尔逊相关，用于衡量**一个二值变量**（如项目是否答对）与**一个连续变量**（如总分）之间的相关程度。

**通俗解释：**

想象班级考试，双列相关回答的问题是："答对这道题的学生，他们的总分是否普遍更高？"

**含义**：

- 数值越高（通常 > .30），说明该项目能较好地区分高分和低分考生。
- 数值接近 0 表示该项目对高低能力者无区分作用，**可能是无效项目**。
- 负值通常说明项目设计有误（比如评分错误或答案干扰性太大）。

**计算公式（简化形式）**：

\[
r_{pb} = \frac{\bar{X}_1 - \bar{X}_0}{s_X} \sqrt{ \frac{p \cdot q}{n} }
\]

其中：

- \(\bar{X}_1\)：答对该题者的平均总分
- \(\bar{X}_0\)：答错该题者的平均总分
- \(s_X\)：总分标准差
- \(p\)：答对该题比例
- \(q = 1 - p\)

**实用解释**：

- 项目 1（r = .582）区分度较高，说明高能力考生更容易做对；
- 项目 22（r = .174）区分度低，即使它难度大，但不能有效地区分考生；
- 项目如果**正确率中等 + 双列相关高**，是非常理想的项目。

**为什么这很重要？**

好的测验题目应该让高能力者容易答对，低能力者不容易答对。如果一道题所有人都答对或都答错，或者答对答错跟能力无关（靠运气），这道题就没有测量价值。

注：双列相关是**经典测验理论中衡量项目质量最常用的指标**之一。
