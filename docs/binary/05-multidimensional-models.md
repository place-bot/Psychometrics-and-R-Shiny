# 5. 多维模型

## 5.1 为什么需要多维IRT？

多维IRT模型包含两个或更多参数来表示每个人。

**现实需求：**

考虑一个阅读理解测验：

- 需要词汇知识
- 需要语法理解
- 需要推理能力
- 需要背景知识

用单一分数无法充分描述学生的能力结构！

**适用情况：**

- 个体在哪些项目难或易方面存在系统性差异
- 多个能力共同影响项目表现
- 需要同时测量多个构念

**实例：**

应用题：

- 学生A：计算强，理解弱（看不懂题）
- 学生B：理解强，计算弱（看懂但算错）
- 单维模型无法区分这两种情况

## 5.2 多维模型的符号系统

- \(X_{is}\)：个体 \(s\) 对项目 \(i\) 的反应（0 或 1）
- \(\theta_{sm}\)：个体 \(s\) 在维度 \(m\) 上的特质水平
- \(\beta_i\)：项目 \(i\) 的难度
- \(\delta_i\)：项目 \(i\) 的容易度截距
- \(\alpha_{im}\)：项目 \(i\) 在维度 \(m\) 上的区分度
- \(\gamma_i\)：项目 \(i\) 的下渐近线（猜测）

注意：多维模型中常用容易度 \(\delta_i\) 替代难度参数。

**为什么改用容易度？**

在多维空间中，"难度"的概念变得复杂，使用"容易度"更自然。

## 5.3 模型分类

多维模型分为两类：

**探索性模型**：

- 类似因子分析
- 估计多维参数以改善拟合
- 不预设维度含义

**适用场景：** "我知道测验测量多个能力，但不确定是什么"

**验证性模型**：

- 为指定维度估计参数
- 预设项目与维度关系
- 检验理论假设

**适用场景：** "我认为这个测验测量计算和推理两个能力，让我验证一下"
