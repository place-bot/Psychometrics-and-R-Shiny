# 4. 分数变换与解释

## 4.1 量表变换的必要性

### 4.1.1 标准化量表的局限

为什么需要变换？

**IRT 默认量表**：

- 在大多数 IRT 软件中，能力值 \(\theta\) 被假设服从标准正态分布：\(\theta \sim \mathcal{N}(0, 1)\)
- 意味着平均能力为 0，标准差为 1
- 实际输出的能力估计通常位于区间 \([-3, 3]\) 内

**这种量表的缺点**：

- 对教育工作者、学生或政策制定者来说不直观
- 例如，\(\theta = 1.3\) 这样的值很难理解其实际含义
- 没有“满分”、“及格线”等明确标识，难以进行教育决策

**实际需求**：

- **百分制**：更适合教学用途，如 \([0, 100]\) 分数
- **SAT 等标准化考试分数**：如 \([200, 800]\)
- **等级评分制度**：如 1–9 等级、A–F 等级制

## 4.2 线性变换公式

### 4.2.1 基本变换公式

线性变换

**能力参数的线性变换**：

\[
\theta^* = a\theta + b
\]

其中：

- \(a > 0\)：控制标准差（尺度大小）
- \(b\)：控制均值（整体偏移）

**为了使 IRT 模型保持不变，项目参数也必须做相应变换**：

- 区分度变换：\(\alpha^* = \dfrac{\alpha}{a}\)
- 难度变换：\(\beta^* = a\beta + b\)
- 猜测参数不变：\(c^* = c\)

**等价性公式**：

\[
P(\theta^*, \alpha^*, \beta^*, c^*) = P(\theta, \alpha, \beta, c)
\]

这意味着，在新旧量表下，**考生正确答题的概率保持不变**，可比性得到保证。

---

**推导过程：从等价性公式出发**

设原模型概率为：

\[
P(\theta) = c + (1 - c) \cdot \frac{1}{1 + \exp(-\alpha(\theta - \beta))}
\]

新模型设：

\[
\theta^* = a\theta + b
\]

希望满足：

\[
P(\theta^*; \alpha^*, \beta^*, c^*) = P(\theta; \alpha, \beta, c)
\]

展开新模型表达式：

\[
P(\theta^*) = c^* + (1 - c^*) \cdot \frac{1}{1 + \exp(-\alpha^*(\theta^* - \beta^*))}
\]

令 \(c^* = c\)，代入 \(\theta^* = a\theta + b\)：

\[
\frac{1}{1 + \exp(-\alpha^*(a\theta + b - \beta^*))} = \frac{1}{1 + \exp(-\alpha(\theta - \beta))}
\]

要两边指数相等：

\[
-\alpha^*(a\theta + b - \beta^*) = -\alpha(\theta - \beta)
\]

去负号并展开：

\[
\alpha^* a \theta + \alpha^* b - \alpha^* \beta^* = \alpha \theta - \alpha \beta
\]

逐项比较：

- 系数项：\(\alpha^* a = \alpha \Rightarrow \alpha^* = \dfrac{\alpha}{a}\)
- 常数项：\(\alpha^* b - \alpha^* \beta^* = -\alpha \beta\)

推得：

\[
\beta^\* = a\beta + b
\]

---

**直觉理解**：

- 把 \(\theta\) 放大缩小或平移后，若要保持 \(\theta - \beta\) 不变形，必须反向缩放 \(\alpha\)，同步平移 \(\beta\)
- 这是 IRT 的 **尺度不变性**：模型只关心“能力 - 难度”的差值

---

**实际意义**：

- 方便在不同测验之间进行链接（score linking）
- 可将 \(\theta\) 映射为更熟悉的分数系统（如 0–100）
- 保证变换后测验结果具有相同解释力，**不依赖常模群体**

---

### 4.2.2 常见变换示例

实际应用的变换

**T分数量表**：

- 均值为 50，标准差为 10
- 变换公式：\(\theta^* = 10\theta + 50\)
- 项目参数：

\[
\alpha^* = \alpha/10,\quad \beta^* = 10\beta + 50
\]

**SAT分数量表**：

- 均值为 500，标准差为 100
- 变换公式：\(\theta^* = 100\theta + 500\)
- 项目参数：

\[
\alpha^* = \alpha/100,\quad \beta^* = 100\beta + 500
\]

## 4.3 IRT分数的独特解释

### 4.3.1 相对于题目的解释

IRT解释的革命性

**传统CTT模型**：

- 只能说明“你超过了多少人”
- 完全依赖参照样本群体（norm-referenced）
- 难以跨时间或测验直接比较

**IRT模型的解释方式**：

- 不依赖常模，而是相对于题目难度来解释能力
- 例：\(\theta = 1.2\) 表示你有 50% 概率答对难度为 1.2 的题
- 所以 IRT 是“item-referenced”，解释更稳定、直观

---

### 4.3.2 Rasch模型的特殊情况

题目中心化的解释

**Rasch模型设定**：

- 统一将题目难度标准化，使其满足：

\[
\sum_{i=1}^{I} \beta_i = 0
\]

- 即整份测验的平均难度为 0

**直观解释**：

- \(\theta = 0\)：答对平均难度题目的概率为 0.5
- \(\theta = 1\)：答对难度比平均高 1 的题目的概率为 0.5
- \(\theta = -1\)：只能答对比平均难度低 1 的题目（概率 0.5）

---

## 4.4 测验链接与可比性

### 4.4.1 共同题目设计

测验链接的方法

**锚题法（common item equating）**：

- 设计两套测验时，保留一部分“锚题”（相同题目）

例如：

- 测验A包含题目 1–30
- 测验B包含题目 21–50
- 公共部分为题目 21–30

**步骤**：

1. 用锚题拟合出 A 与 B 能力量表之间的线性变换关系
2. 将 B 的能力估计转换到 A 的量表（或反之）
3. 实现不同测验间的能力估计对齐与分数可比

## 4.5 IRT分数报告的创新

### 4.5.1 置信区间报告

概率陈述

**传统报告方式**：

- \(\hat{\theta} = 1.2\)，标准误 \(SE = 0.3\)

**改进方式**：

- “有 68% 的置信度，能力位于 \([0.9, 1.5]\)”
- “有 95% 的置信度，能力位于 \([0.6, 1.8]\)”

**优势**：

- 让非专业用户更直观理解估计的不确定性
- 适合教学反馈、心理测量和政策报告使用

---

### 4.5.2 似然比报告

相对可能性

**示例表述**：

> “基于你的答题模式，你的能力处于 1.0 以上的可能性是 0.0 以下的 10 倍”

**数学表达式**：

\[
LR = \frac{L(\theta > 1.0 \mid \mathbf{u})}{L(\theta < 0.0 \mid \mathbf{u})}
\]

**含义**：

- 强调某一侧的可能性显著高于另一侧
- 有助于对“是否通过、是否掌握”等教育决策做出判断
