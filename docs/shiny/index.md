# R Shiny

这一部分是 R Shiny 入门实践。目标不是写完整企业应用，而是先掌握能运行、能展示、能部署的交互式数据应用。

## 学习顺序

1. [学习路径](intro.md)：先看 7 天学习计划和需要安装的包。
2. [Day 1 Hello World](day1.md)：跑通第一个 Shiny App。
3. [Day 2 布局和主题](day2.md)：整理页面结构和主题。
4. [Day 3 输入控件](day3.md)：学习用户输入。
5. [Day 4 输出展示](day4.md)：展示图表、表格和文本。
6. [Day 5 反应式编程](day5.md)：理解 reactive、observe 和 observeEvent。
7. [Day 6 实战项目](day6.md)：完成一个数据探索工具。
8. [Day 7 部署上线](day7.md)：把应用发布给别人访问。

## 这一部分关注什么

| 主题 | 目标 |
| --- | --- |
| UI | 搭建用户能看懂、能操作的页面 |
| Server | 把输入、数据处理和输出连接起来 |
| 反应式编程 | 让应用随用户操作自动更新 |
| 数据展示 | 输出图表、表格、文本和下载结果 |
| 部署 | 让本地 App 变成可访问的网页应用 |
