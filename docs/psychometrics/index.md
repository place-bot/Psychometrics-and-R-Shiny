# 心理测量基础

这一部分放心理测量的入口概念，重点是经典测量理论（Classical Test Theory, CTT）、项目反应理论（Item Response Theory, IRT）以及为什么现代测量需要基于模型（model-based）的解释。

## 学习顺序

1. [CTT 与 IRT 概览](../ctt_irt_intro.md)：先建立两套测量理论的基本差异。
2. [基于模型的测量](../modelbased.md)：再理解为什么项目反应、潜在特质和模型假设要放在一起讨论。

## 这一部分回答什么问题

- 什么是观察分数（observed score）、真分数（true score）和测量误差（measurement error）。
- 为什么同一个总分在不同项目组合下不一定有相同含义。
- 为什么 IRT 会把项目参数（item parameters）和被试能力（ability, \(\theta\)）同时纳入模型。
- 为什么后面的 IRT、CDM 和评分章节都依赖这部分概念。
