# 5. 概念补充2：分类-总计模型（Divide-by-Total Model）

## 5.1 定义

分类-总计模型是指一种建模方式，其中某一类别、维度或子组的得分被除以总得分或总数，以反映其在整体中的相对比例。该结构常用于多维评分、认知诊断、行为占比分析等场景。

其基本思想是：

\[
\text{比例} = \frac{\text{某类别得分或数量}}{\text{总体得分或数量}}
\]

## 5.2 应用示例

**子维度比例评分**

在人格或心理量表中，有时研究者希望了解某个维度（如“焦虑”）在个体整体反应中的比重：

\[
\text{焦虑比重} = \frac{\text{焦虑维度总分}}{\text{量表总分}}
\]

这种方式可用于比较不同被试在各维度上的相对特征分布。

**响应比例建模**

在响应时间或反应模式分析中，有时研究者统计某类反应在总反应中的比例，例如：

\[
P_i = \frac{\text{快速反应次数}}{\text{总反应次数}}
\]

这类比例型指标可以作为个体行为风格的特征输入，用于聚类、预测等建模任务。

**认知诊断中的掌握比例**

在认知诊断模型（CDM）中，研究者有时计算被试掌握的技能数占总技能数的比例，用于衡量综合掌握水平：

\[
\text{掌握比例} = \frac{\text{被试掌握技能数}}{\text{总技能数}}
\]

这种形式常见于后验分析、能力分类或策略建模中。

## 5.3 与传统模型的比较

| 模型类型 | 建模形式 | 特点 |
| --- | --- | --- |
| 分类-总计模型 | 相对比例建模 | 强调部分/整体的比例结构 |
| 经典IRT模型（如2PL） | 基于潜变量的概率建模 | 建模个体与项目之间的匹配 |
| CDM模型 | 建模知识状态与属性掌握 | 适合诊断性与分类性测量 |

## 5.4 小结

分类-总计模型并不是一个标准的IRT模型名称，而是泛指一类以“局部得分除以总得分”的方式建模的结构。其核心在于提取**结构比例信息**，以反映个体在不同维度、类别或行为类型中的相对特征。

若在特定文献、模型框架或软件输出中遇到该术语，应结合上下文进一步明确其定义与建模目的。
