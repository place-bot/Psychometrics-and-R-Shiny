# 17. 多项IRT模型的特殊考虑

## 17.1 与受试者数量高度相关的问题

**1. 异质样本的重要性：**

强烈建议研究者基于异质样本估计参数，可以规避拟合多项模型的第二个问题。

**2. 每个反应类别都需要有反应：**

研究者必须在每个反应类别中都有项目反应。

零频数问题

如果反应类别的反应为零或很少，估计参数的程序将无法在类别阈值或交叉点之间做出良好估计。

## 17.2 实际遇到的问题

**常见情况：**
我们见过许多样本中受试者不选择特定项目类别的情况

**可能原因：**

- 由于特质的准类别性质
- 某些极端类别很少被选择

## 17.3 解决方案

**方法1：合并类别**

- 将很少选择的类别与相邻类别合并

**方法2：重写项目（更好的方法）**

- 减少类别数量
- 重新设计类别标签

重要注意

在合并类别时需要一些特殊考虑，请参见Andrich (1995)

## 17.4 实际建议

样本规划

- 确保足够的样本量（至少500人）
- 确保样本的异质性
- 预测试检查各类别的反应分布
- 必要时调整项目设计
