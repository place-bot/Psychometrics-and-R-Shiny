# 16. 需要多少受试者？

## 16.1 常见问题

学生和同事经常询问：估计IRT模型参数需要多少受试者？

### 16.1.1 没有标准答案

**原因：**

- 取决于许多因素
- 如项目的区分度如何
- 要估计多少个项目参数

### 16.1.2 蒙特卡罗模拟研究

**Reise和Yu (1990)的研究：**

- 表明使用MULTILOG可以用少至250名受试者估计GRM
- 但他们建议约500人

模拟研究的局限性

- 仅在数据与模拟数据匹配时有用
- 实际数据可能与模拟条件不同

## 16.2 替代方法建议

**基于标准误的准则：**
建议研究者拥有足够的受试者，使参数估计的标准误合理小

**"合理小"的定义：**

- 取决于研究者的测量目标
- 项目参数估计对特质水平估计的影响是许多研究者关心的最终问题

## 16.3 本章数据的启示

从本章结果可以看出，即使在有350名受试者的情况下，GRM中的一些类别阈值参数估计也不理想。
