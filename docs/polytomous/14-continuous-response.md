# 14. 连续反应模型

## 14.1 连续反应格式的使用现状

**社会科学研究中的情况：** 研究者很少使用连续反应格式

**连续反应的例子：**

- 要求受试者在1到100的量表上描述观点
- 要求受试者通过在从高到低锚定的线上做标记来表示观点
- 甚至李克特型项目格式的反应也可视为连续量表的近似值

## 14.2 避免连续反应格式的原因

**主要问题：**

- 由于量表扩展而导致的主观性增加
- 产生反应偏差

## 14.3 IRT对连续反应的处理

**Mellenbergh (1994b)的贡献：**

- 连续反应可以通过将其视为潜在变量的观察表现而轻松地由IRT模型处理
- 在这种情况下，可以使用因素分析模型来理解项目反应
