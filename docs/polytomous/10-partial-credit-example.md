# 10. 概念补充5：示例说明为何“部分计分模型”适用于部分正确的项目

在实际的成就测验中，许多项目无法简单地判定为“对”或“错”。这些项目常常具有**部分正确**的可能性，或包含多个评判维度。这类项目若强行用二元评分，不仅会丢失信息，还可能影响模型拟合效果与被试能力估计。

以下是几个典型示例，说明这类项目为何适合使用**部分计分模型（PCM）**或**广义部分计分模型（GPCM）**建模：

## 10.1 示例 1：数学过程题

**题目：** 化简表达式 \(\frac{2x^2 + 4x}{2x}\)

**评分方式：**

| 回答内容 | 得分 | 解释 |
| --- | --- | --- |
| \(\frac{2(x+2)}{2x}\) | 1 | 展示因式分解，但未完全化简 |
| \(x + 2\) | 2 | 正确完成因式分解与约分化简 |
| \(x\) | 0 | 化简错误 |

本题有清晰的部分正确路径，使用 GPCM 模型可更好建模不同能力水平下的作答表现。

## 10.2 示例 2：阅读理解题

**题目：** 阅读短文后回答：“作者的观点是什么？请提供两个支持证据。”

**评分方式：**

| 回答情况 | 得分 |
| --- | --- |
| 提出观点但缺乏证据 | 1 |
| 提出观点 + 一个支持证据 | 2 |
| 明确观点 + 两个合理证据 | 3 |
| 回答无关或错误 | 0 |

该题考查信息整合能力，得分具有等级性结构，非常适合使用 PCM 建模。

## 10.3 示例 3：写作任务评分

**题目：** 写一段话阐述你是否同意“努力比天赋更重要”这一观点。

评分依据可能包括“表达完整性”“语言结构”“逻辑性”等，采用 0–5 分等级评分制。此类项目本质上具备有序多等级特征，最适合使用**等级反应模型（GRM）**或**PCM**进行刻画。

这些例子共同说明：当项目允许部分得分或多等级评分时，传统的二元 IRT 模型已不再适用。此时应采用多类别模型，如 PCM 或 GPCM，以更精准地描述反应行为与能力关系。
