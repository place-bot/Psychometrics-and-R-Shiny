# 0. 引言

## 0.1 什么是多项IRT模型？

**IRT是什么？**

- IRT = Item Response Theory（项目反应理论）
- 这是心理测量学中用来分析测验和问卷的一种方法

**什么是"多项"？**

想象一下两种不同的题目：

1. **二分题目**：对/错，是/否（只有2个选项）
2. **多项题目**：非常不同意/不同意/中立/同意/非常同意（有多个选项）

**为什么需要多项IRT模型？**

传统的IRT模型只能处理对错题，但现实中很多心理测验（比如人格测试、态度调查）都使用多个等级的评分，比如：

- 1-5分的李克特量表
- 从"完全不同意"到"完全同意"的选项

## 0.2 背景与目标

**背景问题：**

第4章介绍的IRT模型只能处理二分项目（对错题），但这有个限制：许多心理学家常用的测验格式无法简单地评分为对错。

**实际需求：**

- 很多测量工具，特别是态度和人格评估，包含多个有序反应类别的项目
- 研究者使用这些格式的主要原因：比二分评分提供更多信息且更可靠
- 对于这些多类别项目反应数据，需要多项IRT模型来表示受试者特质水平与特定类别反应概率之间的非线性关系

**本章目标：**

- 不是提供所有现有模型的全面总结
- 而是提供六个较知名多项模型的基本心理测量特性描述
- 对更广泛内容感兴趣的读者可参考van der Linden和Hambleton（1996）

## 0.3 本章要介绍的模型

1. **Graded Response Model (GRM)**
   **等级反应模型**
   Samejima, Fumiko（1969）提出
   是 2 参数逻辑模型（2PL）的扩展，适用于有序多分类题目。
2. **Modified Graded Response Model (M-GRM)**
   **修正的等级反应模型**
   Muraki, Eiji（1992）提出
   在 GRM 基础上引入灵活阈值设定，更适合实际测量中非对称的评分标准。
3. **Partial Credit Model (PCM)**
   **部分计分模型**
   Masters, Geoffrey N.（1982）提出
   是 Rasch 模型（1PL）对多级评分项目的扩展，每个项目可有独立阈值。
4. **Generalized Partial Credit Model (GPCM)**
   **广义部分计分模型**
   Muraki, Eiji（1990）提出
   在 PCM 基础上增加项目区分度参数（类似 2PL），更灵活地建模项目差异。
5. **Rating Scale Model (RSM)**
   **评定量表模型**
   Andrich, David（1978a, 1978b）提出
   对 PCM 的简化，假设所有题目的阈值结构相同，适用于固定评分标准的量表。
6. **Nominal Response Model (NRM)**
   **名义反应模型**
   Bock, R. Darrell（1972）提出
   适用于无序多分类项目，允许各类别有独立的吸引力与区分度参数。

**模型分类：**

- **"间接"模型**：需要两步过程确定受试者在特定类别反应的条件概率（前两个模型）
- **"直接"模型**：只需一个方程描述受试者特质水平与特定类别反应概率的关系（后面的模型）
