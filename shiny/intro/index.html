    R Shiny 快速入门笔记¶

    0. 前言¶

    0.1 这是什么¶

    这是 R Shiny 入门学习笔记。目标是：在一周内掌握构建基础数据展示应用的完整流程。

    不追求复杂，就是先建立可运行、可解释的最小应用。

    0.2 为什么学习 Shiny¶

    • 研究、教学或项目汇报中需要反复展示数据
    • 希望把 ggplot 图表扩展为可交互界面
    • 希望主要使用 R 语言完成交互式数据展示
    • 希望快速构建网页形式的数据展示结果

    0.3 需要哪些基础¶

    • 会基本的 R（知道怎么装包、写函数、画图即可）
    • 会使用 RStudio 或 Positron 等 R 开发环境
    • 了解基本的数据框、向量和绘图语法

    1. 学习计划（7 天学习路径）¶

    Day 1: Hello World¶

    目标： 跑起来第一个 Shiny App

    要学的：

    • 装 Shiny 包
    • UI 和 Server 的职责
    • 怎么运行 App
    • 基本的输入输出

    笔记内容：

    • 最简单的例子
    • 常见报错和解决办法
    • 一个滑块控制直方图的例子

    Day 2: 布局和主题¶

    目标： 让 App 结构更清晰、视觉更统一

    要学的：

    • sidebarLayout() 侧边栏布局
    • tabPanel() 多标签页
    • 基本的 HTML 标签
    • 使用 bslib 或 shinythemes 设置主题

    笔记内容：

    • 几种常用布局
    • 怎么加标题、说明文字
    • 一键换主题

    Day 3: 各种输入控件¶

    目标： 学会用各种输入框

    要学的：

    • sliderInput() 滑块
    • selectInput() 下拉框
    • checkboxInput() 复选框
    • textInput() 文本框
    • fileInput() 文件上传

    笔记内容：

    • 每种控件的例子
    • 怎么获取用户输入的值
    • 实战：做个数据筛选器

    Day 4: 展示结果¶

    目标： 把分析结果展示出来

    要学的：

    • plotOutput() 展示图表
    • tableOutput() 展示表格
    • textOutput() 展示文字
    • downloadHandler() 下载功能

    笔记内容：

    • ggplot2 图表展示
    • DT 包做交互表格
    • 怎么让用户下载结果

    Day 5: 反应式编程基础¶

    目标： 理解 Shiny 的核心概念

    要学的：

    • reactive() 的作用
    • observe() 和 observeEvent() 的区别
    • 什么时候用什么

    笔记内容：

    • 简单解释反应式
    • 常见的常见问题和解决办法
    • 实战：联动的下拉框

    Day 6: 实战项目¶

    目标： 做个完整的小项目

    项目选择（选一个）：

    1. 数据探索工具：上传 CSV，自动生成统计图表
    2. 简单仪表板：展示一些关键指标，定时刷新
    3. 数据对比工具：上传两个文件，对比差异

    笔记内容：

    • 完整代码
    • 遇到的问题和解决过程
    • 可以改进的地方

    Day 7: 部署上线¶

    目标： 让别人能访问你的 App

    要学的：

    • shinyapps.io 入门部署
    • 本地局域网分享
    • 打包给别人用

    笔记内容：

    • 注册账号和部署步骤
    • 常见部署错误
    • 分享链接给别人

    2. 资源整理¶

    2.1 必装的包¶

    # 核心
install.packages("shiny")

# 主题与 Bootstrap 样式
install.packages("bslib")

# 更多控件
install.packages("shinyWidgets")

# 交互式表格
install.packages("DT")

# 交互式图表（可选）
install.packages("plotly")

    2.2 推荐资源¶

    • 官方教程：https://shiny.posit.co/r/getstarted/shiny-basics/lesson1/
    • 控件展示：https://shiny.posit.co/r/gallery/widgets/widget-gallery/
    • Stack Overflow：遇到问题就搜 "shiny + 你的问题"

    2.3 速查表（Cheat Sheet）¶

    最常用的函数

    UI 端：

    • fluidPage() - 创建页面
    • sidebarLayout() - 侧边栏布局
    • XXXInput() - 各种输入
    • XXXOutput() - 各种输出

    Server 端：

    • renderPlot() - 渲染图表
    • renderTable() - 渲染表格
    • reactive() - 创建反应式表达式
    • observe() - 观察变化

    3. 总结¶

    3.1 适用对象¶

    • 只想快速上手，优先完成入门实践的
    • 有个简单需求要实现的
    • 需要快速完成数据展示原型的
    • 希望以短教程形式学习的

    3.2 不适用对象¶

    • 要做复杂企业应用的
    • 追求最佳实践的
    • 要深入理解原理的
    • 要处理大规模数据的

    3.3 学习成果¶

    • 数据上传和展示
    • 简单的交互式图表
    • 基本的数据筛选和下载
    • 够用的界面布局

    学习建议

    1. 第一次肯定会遇到很多报错，正常
    2. 从最小可运行示例开始，再逐步修改参数和结构
    3. 不要纠结原理，先完成可运行示例，再逐步理解原理
    4. 遇到问题时优先阅读报错信息、官方文档和可复现示例，再查阅社区问答
